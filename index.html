1. Find the missing number in the array (Array에서 빠진숫자 구하기)

You are given an array of positive numbers from 1 to n, such that all numbers from 1 to n are present except one number x. You have to find x. The input array is not sorted. Look at the below array and give it a try before checking the solution.

let find_missing = function(input) {
// calculate sum of all integers
// in input list

let actual_sum = 0;
for (let i in input) {
actual_sum += input[i];
}
// There is exactly 1 number missing
let n = input.length + 1;
let sum_of_n = Math.floor((n * (n + 1)) / 2);
return sum_of_n - actual_sum;
};

2. Determine if the sum of two integers is equal to the given value (Array안에 두 숫자의 합인 숫자가 들어있는지)

Given an array of integers and a value, determine if there are any two integers in the array whose sum is equal to the given value. Return true if the sum exists and return false if it does not. Consider this array and the target sums:

let findSumOfTwo = function(A, val) {
let foundValues = new Set();
for (let a in A) {
if (foundValues.has(val - A[a])) {
return true;
}

foundValues.add(A[a]);
}
return false;
};

let v = [5, 7, 1, 2, 8, 4, 3];
let test = [3, 20, 1, 2, 7];

}
